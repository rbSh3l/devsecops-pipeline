### 🐳 DevSecOps CI/CD Security Pipeline

Built a Docker-based DevSecOps pipeline using GitHub Actions, Gitleaks, Semgrep, and Trivy.

#### Key Features

* 🔐 Secret Detection (Gitleaks)
* 🔎 Static Application Security Testing (Semgrep)
* 📦 Docker Container Vulnerability Scanning (Trivy)
* ⚙️ Automated Security Validation via GitHub Actions
* 🛡 GitHub Code Scanning Integration
* 🚨 Detection of Critical and High Severity Vulnerabilities

**Skills:** Docker, GitHub Actions, DevSecOps, CI/CD, Vulnerability Management, SAST, Container Security, Security Automation
